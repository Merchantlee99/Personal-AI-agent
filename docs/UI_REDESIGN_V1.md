# Personal AI Agent UI Redesign v1

## 목적
- `Morpheus / Clio / Hermes` 3인 에이전트 콘솔을 단일 화면에서 운영 가능하게 만든다.
- 대화/알림/상태를 분리하면서도, 활성 에이전트 컬러를 전체 패널에 일관되게 반영한다.
- 몰입형 중앙 시그널(홀로그램 스타일)과 실사용 가능한 입력 UX를 동시에 제공한다.

## 현재 반영된 레이아웃
- 좌측 `SYSTEM STATUS` 패널: 에이전트 선택, 상태 점, 미확인 알림
- 중앙 시그널: 에이전트별 기하 도형 + 동심원 + 코어 에너지 애니메이션
- 우측 `API BUDGET` 패널: provider별 사용량/잔여량
- 하단 `CONVERSATION` 패널: 메시지 스트림 + 자동 확장 입력창 + 인필드 전송 버튼

## 시각 시스템
- 활성 에이전트 기준 컬러 동기화
  - Morpheus(ace): `#4338CA / #6366F1`
  - Clio(owl): `#EA580C / #F97316`
  - Hermes(dolphin): `#059669 / #10B981`
- 적용 대상
  - 좌/우 패널 배경 틴트, 보더, 글로우
  - 하단 대화 패널 보더/그라데이션/포커스 링
  - 입력창 포커스, 전송 버튼 상태, 스크롤바 톤
- 전환 애니메이션: `0.5s ease`

## 상호작용 UX
- 대화창 토글
  - 접힘 상태에서 패널 클릭 시 열림
  - 상단 중앙 핸들바 클릭으로 접기/펼치기
- 대화가 길어지면 하단 패널이 자동으로 위로 확장
- 입력창
  - `Shift+Enter`: 줄바꿈
  - `Enter`: 전송
  - 입력량에 따라 자동 높이 조절
  - 전송 버튼은 입력창 내부 우측 하단 고정
  - 클릭/엔터 전송 시 짧은 pressed 피드백 제공

## 데이터 연동 포인트
- `/api/chat`: 수동 대화 응답
- `/api/agent-updates`: proactive 메시지 폴링/표시
- `/api/hermes/daily-briefing`: Hermes 자동 브리핑 큐잉
